WinaeroTweaker

Jun 05, 2024
阅读时长: 1 分钟

Winaero Tweaker 是我非常喜欢的 Windows 工具
这个工具能对很多 Windows 的功能进行微调,一开始认识他是因为当时想要找一个能禁用 Windows 自动更新的工具。

因为它的功能实在是特别的多，我推荐几个我觉得很有用很喜欢的功能：

Disable Windows Update

注意关闭 Windows 更新的同时也不能在商店下载东西了,但是可以随时取消勾选来恢复

restart explorer

桌面右键可以重启explorer

take ownership

一键一键获取文件(夹)的读写权限

Windows terminal

可以增加加非常有用的右键菜单,比如一键从 wsl 中开启!
附一个 wsl 安装命令 wsl --install

shut down menu

增加一个右键关电脑菜单,提升你的生活品质

icon cache

增大你的图标缓存,会让电脑感觉变快一些 有的时候有的图标变成白色，可以用这个按钮强制清除一下缓存

UAC

关闭 uac 减少电脑对你的打扰

short

关闭拖放快捷方式时候产生的" - 快捷方式" 文件名称后缀
修改快捷方式图标的样子，比如你可以关闭它的小箭头这样让桌面看起来会很干净
