我自用多年的导航页 WatchTV

Apr 25, 2026
阅读时长: 1 分钟

因为一些原因我把它放在隐私仓库里了，所以我把一个当前版本的不包含任何我自己配置的程序本体分享一个可以在下面的链接下载：

https://kasusa.lanzoul.com/b0j1evt2h 密码:fmca

使用方法

  1. 安装node，建议使用nvm安装（上方链接内可下载 windows 版），安装完成后使用nvm安装node
nvm install  20.19.0
nvm use 20.19.0
  1. 下载上方链接内的 watchtv，解压后进入文件夹，安装依赖
npm install
  1. 运行
node server.js
  1. 打开浏览器访问 http://localhost:8901 就可以看到导航页了

打开后台

打开浏览器访问 http://localhost:8901/tvmin.html 密码是 admin 就可以进入后台了(密码可以在 password.txt 里修改)

你也可以在页面中按下 ctrl + alt + m 来进入后台.

其他

  • 它的设计主要是依赖于图标,没有显示名字的区域，可能需要熟悉一段时间才能适应。
  • 相对于大多数导航页来说它的设计更加的简洁， 并且配置非常简单。
  • 它拥有右键菜单设计，你可以对一个图标右键来进入它的多种相关页面（比如说 b 站你可以进入他的投稿、搜索、电影）。
  • 后台设计让你可以通过他的后台来快速的添加icon、修改图标以及链接
  • 它拥有手机端兼容性，保证手机端像电脑端一样使用起来很简洁。
