install sougou for ubuntu

you can follow this guide

download and install

download sougou here

fix every thing sougou need

sudo apt install fcitx
sudo cp /usr/share/applications/fcitx.desktop /etc/xdg/autostart/
sudo apt purge ibus
sudo apt install libqt5qml5 libqt5quick5 libqt5quickwidgets5 qml-module-qtquick2
sudo apt install libgsettings-qt1

install the package again

sudo dpkg -i sougou...deb

reboot

settings and stuff

gotoubuntu settings - Region & Language - Manage installed Languages

choose fcitx

goto fcitx configuration , search for sougoupinyin and add it
