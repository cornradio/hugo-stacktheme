Cloudflare Pages 免费部署静态博客

Nov 22, 2024
1 minute read

为什么要换

因为一些费用的原因，我打算把我的服务器从vultr 上面给取下来，这样我就可以每个月省下来五刀左右的费用，相当于35块钱 但是同时我又不想放弃我这个博客,于是目前的做法是使用免费的 cloudflare pages

使用 cf pages

进入 cloudflare ， 左侧往下滑，找到 workers-and-pages ，选择 page。 page 可以通过从 github 同步 repo ， 或者直接上传静态 html 文件。

我选择同步 github repo ， 需要进行授权操作。
