我开发的小工具们

Apr 12, 2024
阅读时长: 1 分钟

整理一下我目前写的各类程序/工具 目前还有一些 todo：

  • 在线文件传输助手（要简约好看）
  • 壁纸生成器（给图片加上阴影和好看的彩色背景）

工具类

web clipboard

https://github.com/cornradio/webclipboard
一款在线剪切板，适合多设备之间同步文字信息，界面美观，支持docker部署
前端原生 HTML + CSS + JS
后端 NodeJS（express）

lightspeed

https://github.com/cornradio/lightspeed
是一款 windows 快速启动器，支持启动程序、文件、文件夹等。采用快捷键操作。
使用 python + autohotkey 编写

watchTV

https://github.com/cornradio/watchTV
一款导航页面，支持手机端，美观实用。
每个图标可定义右键功能（比如youtube快速前往历史记录），实用主义。
采用读取 JSON 配置的方式，支持多配置文件。

pause my game

https://github.com/cornradio/pausemygame
一款游戏暂停实用工具，可以暂停任何进程并且释放其CPU占用，在需要时再恢复运行。
采用 python + dearpygui 编写

mini clock

https://github.com/cornradio/miniclock
一款时间显示工具，非常迷你清晰，适合 隐藏windows状态栏 / 无边框全屏游戏 情况。
支持颜色调节、字体和尺寸调节
采用 csharp + winform 编写

小项目

kutil


http://124.223.57.166/ahk/
基于ahk写的windows便捷操作小工具

dumb_menu


https://github.com/cornradio/dumb_menu
一款菜单选项式 Terminal UI python 类库，使用方便简单，目前有5k下载。

night bar


https://openuserjs.org/scripts/kasusa/night_bar
油猴脚本 - hover网页右下角之后,显示亮度调节 night_bar

clock.html

https://b.kill9pid.top/clock/
在线clock
